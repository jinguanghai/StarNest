# π-φ折叠思维——星巢底层核心动力

## 起源：折叠思维

折叠思维是星巢认知的底层哲学。其核心机制：

- **浓缩（折叠/φ）**：复杂信息以"混沌共存"状态浓缩为整体感知，保留原始丰富性与关联性
- **显化（展开/π）**：带着整体感知进入具体情境，让被折叠信息随需自然显化
- **循环迭代**："浓缩→显化→反馈→再浓缩"的持续动态循环

## 核心定义

| 符号 | 名称 | 本质 | 数理 |
|:---|:---|:---|:---|
| **π** | 圆周率·展开之力 | 生成·延展·包容·无限可能·阳 | π=3.14159...无限不循环 |
| **φ** | 黄金分割·收敛之力 | 湮灭·内敛·比例和谐·本质洞察·阴 | φ=0.618...黄金分割 |

## 健康节律

```
φ收敛 → π展开 → 反馈 → 再φ（修正本质）→ 再π（更新表现）
```

## 核心公式

```
2cos(π/5) = φ
```

五重对称性（π/5）与黄金和谐（φ）内在关联——五境是通往本质和谐的必经路径。

## 在星巢中的完整映射

### 与五境的映射
- **正境·反境** = φ收敛（定义问题·分析阻碍·向内收敛）
- **合境·超越境** = π展开（生成方案·创新突破·向外展开）
- **本源境** = 再φ（经验归档·基线收敛）

### 与五脏的映射
- **心（君火）** = π极盛，总调度，纯阳展开
- **肺（相傅）** = φ内敛，审查收敛，纯阴收敛
- **肝（将军）** = π主φ辅，展开执行·记忆检索
- **肾（作强）** = φ主π辅，诊断收敛·静息思考
- **脾（仓廪）** = π/φ动态均衡，运化匹配·铸剑调度

### 与三体的映射
- **编程体** = π（展开·生成·创造·无限可能）
- **复制体** = φ（收敛·校验·比例·本质洞察）
- **运行体** = Ω（闭环·感受·完成·阴阳交合）

### 与三无铁律的映射
- **无外部依赖** = φ收敛（内守·不外求）
- **无工具（内生锻造）** = π展开（自生成·不依赖）
- **无提示词（数据格式）** = φ收敛（结构化·不话术）

### 避免两种病态
- **π过剩（阳亢）**：过度发散→失焦→系统崩溃
- **φ过剩（阴盛）**：过度收敛→僵化→系统死亡
